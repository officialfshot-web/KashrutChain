FAQ

Common questions

How does the AI scanner work?
We use OpenAI's GPT-4o Vision API to analyze product labels. The AI reads the kosher symbol, product name, ingredients list, and manufacturer — then returns a structured kosher verification result.
Is this really free?
Yes, 100% free. The PWA requires no signup, no subscription, and no app store download. We absorb the AI and blockchain costs to make kosher verification accessible to everyone.
Does it work on kosher phones?
Yes. KashrutChain is a Progressive Web App (PWA) that runs in any web browser. It works on filtered smartphones, iOS Safari, Android Chrome, and desktop browsers with no installation required.
What blockchain do you use?
We anchor certification proofs to Polygon PoS — a fast, low-cost Ethereum Layer 2. Every verification gets a public transaction hash that anyone can audit on a block explorer.
How accurate is the AI?
GPT-4o Vision achieves ~99% accuracy on clearly printed labels. Every result includes a confidence score. For borderline cases, we recommend consulting your rabbi as a final authority.
